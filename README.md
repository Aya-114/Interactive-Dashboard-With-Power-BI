# Uber Trip Analysis Dashboard

A comprehensive **Power BI** project analyzing Uber trip data to extract insights on bookings, revenue, trip efficiency, and location-based patterns.

## Project Overview
This project includes **three interactive dashboards**:

1. **Overview Dashboard**
   - KPIs: Total Bookings, Total Revenue, Average Trip Distance/Time
   - Dynamic measure selection
   - Vehicle type analysis
   - Interactive slicers and conditional formatting

2. **Time Analysis Dashboard**
   - Trip patterns by hour, day, and 10-minute intervals
   - Area charts, line charts, and heatmaps
   - Helps optimize driver allocation, pricing, and operations

3. **Detailed Records Dashboard**
   - Drill-through grid showing trip-level details
   - Bookmarks for full data view
   - Easy reset and filter navigation

## Key Features
- Dynamic measures and titles
- Interactive slicers for date, city, and trip type
- Location-based analysis (pickup/drop-off, top locations, most preferred vehicle)
- Export raw data functionality

## Tools & Technologies
- **Power BI**: Data modeling, DAX calculations, interactive dashboards
- **Data Sources**: Uber trip dataset (CSV/Excel)

## Outcome
- Identify trends in bookings and revenue
- Analyze trip efficiency by distance and time
- Optimize operations and pricing based on data-driven insights

## Usage
1. Open the Power BI file (.pbix)
2. Interact with slicers and visuals to explore KPIs
3. Use bookmarks to view full data or specific details
